# Prompt 25: Build UX/UI Foundation

## Mục tiêu
Dựa trên các file thiết kế `ux-flow-overview-v1.png` và `s-budget-ui-v1.pdf`, xây dựng nền tảng UX/UI (foundation) cho S.Budget Mobile App. Thiết lập ứng dụng bằng React Native (Expo), cấu hình các thư viện cốt lõi và xây dựng flow điều hướng (navigation).

## Tasks
1. Khởi tạo frontend mobile app với tên `mobile-app` (nếu chưa có) sử dụng Expo, TypeScript.
2. Cài đặt và cấu hình NativeWind (TailwindCSS cho React Native), React Navigation, Zustand, TanStack Query.
3. Cấu hình màu sắc, typography, spacing chuẩn theo tài liệu thiết kế.
4. Xây dựng Navigation Flow cơ bản (Auth Stack, Main Tab Navigator với Dashboard, Transaction, AI Assistant, Profile).
5. Tạo các màn hình rỗng (dummy screens) để test flow và navigation theo UX mới.

## Ngữ cảnh
S.Budget hiện tại đã có Backend (Microservices) hoàn thiện đến phần Unit Test. Phase hiện tại tập trung vào UX/UI Foundation cho Mobile App.
